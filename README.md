# EyeVacs Web Admin

EyeVacs 광고 플랫폼의 관리자/사업자용 웹 어드민 대시보드입니다.

## 주요 기능

### 사업자 모드
- **내 가게 관리** — 가게 정보 등록 및 수정
- **내 광고 관리** — 광고 등록, 수정, 삭제
- **내 쇼핑 관리** — 쇼핑 아이템 등록 및 관리
- **통계** — 광고/쇼핑 성과 통계 확인
- **요금제** — 요금제 확인
- **FAQ** — 자주 묻는 질문

### 관리자 모드
- **통계 대시보드** — 전체 가게/광고/쇼핑 현황
- **가게/광고/쇼핑 조회** — 전체 데이터 열람
- **영업 관리** — 리드 관리 및 영업 현황
- **리포트** — 종합 리포트

## 기술 스택

- React 19 + TypeScript
- Tailwind CSS
- Headless UI
- Heroicons

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

`http://localhost:3000`에서 확인할 수 있습니다.

## 빌드

```bash
npm run build
```

`build/` 폴더에 프로덕션 빌드가 생성됩니다.
